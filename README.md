# Lucas Dev

This project was based in NextJS Framework.
## Stack

**Front-end:** React, Styled Components

**Back-end:** NextJS


## Run local

Clone the project

```bash
  git clone https://github.com/LucasRichter/my-dev-site
```

Enter in the project folder

```bash
  cd my-dev-site
```

Install the dependicies

```bash
  yarn install
```

Start the server

```bash
  yarn start
```
